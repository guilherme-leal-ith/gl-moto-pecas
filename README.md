# Projeto: Banco de Dados - Loja de Peças Automotivas

Este projeto foi desenvolvido como parte da disciplina de **Modelagem de Dados**, com o objetivo de aplicar conceitos de modelagem relacional, criação de tabelas, relacionamentos e manipulação de dados utilizando SQL.

A proposta do trabalho foi modelar o banco de dados de uma **loja de peças de automóveis**, contemplando entidades como clientes, funcionários, produtos, vendas, fornecedores, estoque e pagamentos.

Durante o desenvolvimento, foram aplicados conceitos como:
- Criação de tabelas (DDL)
- Inserção, atualização e remoção de dados (DML)
- Consultas SQL (DQL)
- Uso de chaves primárias e estrangeiras
- Relacionamentos entre entidades
- Normalização básica

## Próximos Passos

A ideia agora é evoluir este projeto acadêmico para algo mais completo.

Estamos iniciando a **refatoração do sistema** com o objetivo de transformá-lo em uma aplicação **fullstack**, utilizando:

- **Frontend:** React  
- **Backend:** Java com Spring Boot  
- **Banco de Dados:** MySQL  

O objetivo é construir uma aplicação funcional que permita gerenciar vendas, produtos, clientes e estoque de forma integrada.

## Banco de Dados

O banco de dados foi desenvolvido utilizando **MySQL**, com foco em organização, integridade referencial e boas práticas de modelagem relacional.
